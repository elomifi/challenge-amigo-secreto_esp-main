<h1 align="center"> ENCUENTRA A TU AMIGO SECRETO  </h1>

Este proyecto elige a tu amigo secreto de una lista de amigos que tú le des.

## :hammer:Funcionalidades del proyecto

- `Funcionalidad 1`: Permite agregar amigos a la lista.
- `Funcionalidad 2`: Muestra la lista de amigos.
- `Funcionalidad 3`: Realiza el sorteo del amigo secreto y lo muestra.
